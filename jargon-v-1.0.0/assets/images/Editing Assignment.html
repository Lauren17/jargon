Lab 3:  Custom Post Type
Weight: 15%
Due : November 10th.


Custom Post Type Loading Interface 
Due In Class Nov 7th.

INTRODUCTION

Over the last few weeks we have been building the base templates for a WordPress site. In this assignment you have to create a custom post type for the articles section that we build in lab 1 using the Custom Post Type UI  and the Advanced Custom Fields plugins. You have two choices for this project you can work on the jargon site that was lab one and recreate it in WordPress for the reset of the term or work on your own custom WordPress site with the same build features

BUILD

If your using dmitstudent.ca as your server for WordPress you must create a new WordPress site on the server in the dmit2032 folder. You do not want to mix your development site labthree with the final build which will be in the jargon folder.  If you have your own hosting make sure you have two seperate sites one for class and one for the final site your building. 


Task 1:  Install  New WordPress Site on dmitstudent.ca
    -  create a new database
    -  create a folder in called jargon inside the dmit2032 folder .
   -  login button right (login to wp-admin dashboard)

Task: 2 Nav Bar
  -  jargon logo left
  -  login button right (login to wp-admin dashboard)

Task 3: Header
  -  site name
  -  site tagline
  -  site branding logo?
  - background image (either css background images or using the image tag implement responsive image best practices)
  - call to action button  -  custom builds should have these five assets as a minimum.

Task 4: Featured Articles Section
  -  Four featured articles including title and tagline and any other media assets that you think may enhance the aesthetic appearance of the jargon site.
  -  For the featured articles create 3 categories css ui/ux and optimization 
  -  Create 4 articles for each category
  -  After the featured articles section create a section called categories 
  -  Create three categories with the heading css ui/ux and optimization
  -  For each of the categories create links to each of the four articles in that category.

Task 5: Custom Post Type
  -  Create a loading interface for the articles custom post type.
  -  The included pdf file is an example of the minimal layout for an article in any category.
